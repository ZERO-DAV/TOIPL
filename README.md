# The Observatory (TOIPL)

## Interactive Physics Laboratory

TOIPL is a modular, open-source physics engine designed to simulate 25+ laws across 7 scientific branches. Built with a "Modular Core" architecture, it ensures high performance and scalability, featuring a tactical UI optimized for precision and fast data access.

---

## Tech Stack
* **Engine:** Vanilla JavaScript (ES6+).
* **Rendering:** Canvas API for real-time physics simulations.
* **UI/UX:** CSS Variable-driven, high-density terminal-style layout.
* **Architecture:** Decoupled, module-based system for seamless expansion.

---

## Features
* **Modular Design:** Add new physics laws easily without modifying the core engine.
* **Tactical UI:** Minimalist, high-efficiency interface designed for engineers.
* **High Performance:** No external libraries or frameworks; purely Vanilla JS.
